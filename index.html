Payout APIs

Simple fund transfer APIs to automate your payouts flow

FUND TRANSFER API

Call Fund transfer API with the unique request id, beneficiary details, mode and amount.
Fund Transfer API Request

{

"requestId" : "pout202008260002",

"beneficiaryName" : "Demo Company",

"accountNo" : "900900900900",

"ifscCode" : "UTIB0002951",

"amount" : 5000,

"mode" : "IMPS",

"purpose" : "For Invoice 20220200789 ",

}

STATUS API

Call the status API with request id to know the fund transfer status.
Fund Transfer Status API Response
